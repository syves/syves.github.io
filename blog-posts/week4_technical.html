What does map do?

  • For each element in your collection map transforms the element and returns it in a new array

  • .map returns a new array with your transformed collection

      • my_friends = ["sue", "jess", "kim"]

      • my_friends_caps = my_friends.map {

      • |name| print "#{name.upcase}!" }

      • => returns ["SUE!", "JESS!", "KIM!]

      • No mutation!

      • print my_friends = ["sue", "jess", "kim"]

      • .each changes the element temporally and returns the original collection passed to it.

      • ["sue", "jess", "kim"].each {

      • |name| print "#{name.upcase}! " }

      • temporarily changes values to "SUE! JESS! KIM!

      • => returns ["sue", "jess", "kim"]

      Why should I use it?

      • .map does not mutate your original collection

      • .map always returns an array
