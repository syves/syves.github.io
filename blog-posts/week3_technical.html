Why is JavaScript so popular?

because its popular

  • Many people have adopted javaScript over the years. Those people made many libraries, books and documents to support its use, which in turn leads to more use.

  • Well, misery loves company. There is a large community of people to work and collaborate with and many of them are very helpful!

  • It is ubiquitous. JS in the backend and if you haven't noticed, it's the scripting language of the web.

  • It is a flexible language, with few rules, I can write in the style I want to write in. I have a lot of control over my code, especially if I use the underscore library

  • because it's on the browser it is fast.

  • Easy to learn and use

  • It is compatible with other languages

  • Basically it's a dynamic object-oriented general-purpose programming language. And if you like to avoid objects it can be fairly functional. I got by without objects until today!

What is wrong with Js?

because it's on the browser

  • It can not access databases

  • Can not read from or write to client files

  • Cannot access pages on other domains

  • Cannot protect pages sources or images

  • Yikes! It is interpreted differently by different browsers

  • Lack of security
